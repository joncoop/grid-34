Basic template

The fastest way to get started is to simply copy the HTML template below. There is no need to download anything as odin.css can be served from the jsDelivr CDN. If you are integrating Odin into an existing project, make sure that you are including an HTML5 doctype declaration and a viewport meta tag so that your site will have proper responsive behaviors.

Any resets such as normalize.css should be linked before odin.css. Custom stylesheets and any stylesheets that override default Odin styling should be linked after. You will also need to link to odin.js and JQuery if you plan on using slideout navbars or carousels. Otherwise, the JavaScript referrences can be deleted.

  
<!DOCTYPE html>

<html lang="en">

<head>
  <!-- metadata -->
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">

  <!-- page title -->
  <title>Page Title</title>

  <!-- styles -->
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/joncoop/odin@{{ site.version }}/dist/css/odin.min.css">

  <!-- link to your custom stylesheets here -->

</head>

<body>

  <!-- make your grid and add content here -->





  <!-- include any javascript below this line -->
  <script src="https://code.jquery.com/jquery-3.5.1.min.js"></script>
  <script src="https://cdn.jsdelivr.net/gh/joncoop/odin@{{ site.version }}/dist/js/odin.min.js"></script>

</body>

</html>

Download

If you'd rather have the source code stored locally, then you can download a template to get started. The framework is defined entirely in odin.css. The main.css file just contains a somewhat organized template for adding any custom CSS or applying overrides to Odin's default styling. The index.html file is already linked to each CSS and JavaScript file. None of the CSS or JavaScript in the dowlnoad is minified in case you want to mess with it.

  
dist/
├── css/
│   ├── main.css
│   └── odin.css
├── js/
│   └── odin.js
└── index.html
