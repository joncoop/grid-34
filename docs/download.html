Download

The repository contains a dist folder which has the files you need to get started. The framework is defined entirely in odin.css. The main.css file just contains a somewhat organized template for adding any custom CSS. It is not a part of the framework. Normalize is useful, but not required. It is included in the repository for convenience.

  
    dist/
    ├── css/
    │   ├── main.css
    │   ├── normalize.css
    │   └── odin.css
    └── index.html

The fastest way to get started is simply to copy the dist folder and start working.

If you are integrating Odin into an existing project, then copy odin.css into your project directory. Any resets such as normalize.css should be linked before odin.css. Custom stylesheets should be linked last.

Only the responsive navbar element requires odin.js and it is dependent on JQuery (not included). Hopefully, a future version will remove the JQuery dependency.
