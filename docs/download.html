The files

The download contains all the files you need to get started. The framework is defined entirely in odin.css. The main.css file just contains a somewhat organized template for adding any custom CSS or applying overrides to Odin's default styling. Responsive navbars and carousels make use of odin.js. Both are dependent on JQuery (not included). The index.html file is already linked to each CSS and JavaScript file.

  
    odin-template/
    ├── css/
    │   ├── main.css
    │   └── odin.css
    ├── js/
    │   └── odin.js
    └── index.html

Getting started

The fastest way to get started is simply to download the template and start working.

To integrate Odin into an existing project, download the temlate and copy odin.css into your project directory. Any resets such as normalize.css should be linked before odin.css. Custom stylesheets and any stylesheets that override default Odin styling should be linked after. You will also need to link to odin.js and JQuery if you plan on using responsive navbars or carousels.

If you're interested in the nitty-gritty of how I developed this framework, then check out the project on GitHub.
