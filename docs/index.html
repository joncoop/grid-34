Name of Grid System

Overview

This is a mobile-first grid framework. Therefore, classes prefixed by sm apply to small and larger devices, md to medium and larger, and lg to large only.

Mixed width columns

Responsive columns

In this example, the columns will be full width on small devices but will each span one-third of the container on medium devices and larger.

col-sm-12 col-md-4
col-sm-12 col-md-4
col-sm-12 col-md-4
    
      <div class="row">
        <div class="col-sm-12 col-md-4">1</div>
        <div class="col-sm-12 col-md-4">2</div>
        <div class="col-sm-12 col-md-4">3</div>
      </div>

Here's how to have different column flows for all three device widths.

col-sm-12 col-md-6 col-lg-3
col-sm-12 col-md-6 col-lg-3
col-sm-12 col-md-6 col-lg-3
col-sm-12 col-md-6 col-lg-3
    
      <div class="row">
        <div class="col-sm-12 col-md-6 col-lg-3">1</div>
        <div class="col-sm-12 col-md-6 col-lg-3">2</div>
        <div class="col-sm-12 col-md-6 col-lg-3">3</div>
        <div class="col-sm-12 col-md-6 col-lg-3">4</div>
      </div>

Columns will be full width on small devices by default and the col-sm-12 class can be omitted if desired. Therefore, the following code will produce the same result as the example above.

    
      <div class="row">
        <div class="col-md-6 col-lg-3">1</div>
        <div class="col-md-6 col-lg-3">2</div>
        <div class="col-md-6 col-lg-3">3</div>
        <div class="col-md-6 col-lg-3">4</div>
      </div>

Responsive helper classes

Responsive images

Responsive elements are given a max-width of 100% so that they will not overflow their containers. You can observe responsive images in action by resizing the browser window as you look at the example below. The responsive class can be applied to img, video, and iframe elements only.

    
      <img class="responsive" src="example_image.jpg" alt="">

Other Helper Classes

Hide/show content

Helper classes are provided to hide/show content based on the size of the viewport. They can be applied to any element. Usage should be fairly obvious. Resize the browser window to see a demonstration.
