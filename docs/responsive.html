Responsive helper classes

Default responsive elements

The elements img, iframe, and video are given a max-width of 100% by default so that they will not overflow their containers. It is not necessary to apply any extra classes to make these elements responsive.

Hiding content

The following helper classes are provided to content on a given viewport size and down along with a given viewport size and up: hide-xs-down, hide-sm-down, hide-md-down, hide-sm-up, hide-md-up, and hide-lg-up. They can be applied to any element. Usage should be fairly obvious. Resize the browser window as you view the sentences below to see a demonstration.

This sentence is visible on the following viewports: sm, md, lg.
 This sentence is visible on the following viewports: md, lg.
 This sentence is visible on the following viewports: lg.
 This sentence is visible on the following viewports: xs.
 This sentence is visible on the following viewports: xs, sm.
 This sentence is visible on the following viewports: xs, sm, md. 

  
    <p class="hide-xs-down">This sentence is visible on the following viewports: sm, md, lg.</p>
    <p class="hide-sm-down">This sentence is visible on the following viewports: md, lg.</p>
    <p class="hide-md-down">This sentence is visible on the following viewports: lg.</p>

    <p class="hide-sm-up">This sentence is visible on the following viewports: xs.</p>
    <p class="hide-md-up">This sentence is visible on the following viewports: xs, sm</p>
    <p class="hide-lg-up">This sentence is visible on the following viewports: xs, sm, md.</p>

You can combile classes to show content on a specific range of viewports. There are no classes to hide or show content on specific viewports. You'll need to write custom classes if you want to hide or show content on a non-contiguous range of viewports.

This sentence is only visible on the following viewports: sm, md.

  
    <p class="hide-xs-down hide-lg-up">This sentence is only visible on the following viewports: sm, md.</p>
