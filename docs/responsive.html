Responsive helper classes

Default responsive elements

The elements img, iframe, and video are given a max-width of 100% by default so that they will not overflow their containers. It is not necessary to apply any extra classes to make these elements responsive.

Hide/show content

The following helper classes are provided to hide/show content based on the size of the viewport: hide-xs, hide-sm, hide-md, hide-lg, show-xs-only, show-sm-only, show-md-only, and show-lg-only. They can be applied to any element. Usage should be fairly obvious. Resize the browser window as you view the sentences below to see a demonstration.

This sentence is not visible on extra-small viewports.

This sentence is not visible on small viewports.

This sentence is not visible on medium viewports.

This sentence is not visible on large viewports.

This sentence is only visible on extra-small viewports.

This sentence is only visible on small viewports.

This sentence is only visible on medium viewports.

This sentence is only visible on large viewports.

  
    <p class="hide-xs">This sentence is not visible on extra-small viewports.</p>
    <p class="hide-sm">This sentence is not visible on small viewports.</p>
    <p class="hide-md">This sentence is not visible on medium viewports.</p>
    <p class="hide-lg">This sentence is not visible on large viewports.</p>
    <p class="show-xs-only">This sentence is only visible on extra-small viewports.</p>
