Mixed width columns

It's totally fine to mix column widths. Just make sure that the sum of the column widths is 12 for each row.

Responsive columns

asdf

In this example, the columns will be full width on small devices but will each span one-third of the container on medium devices and larger.

col-sm-12 col-md-4
col-sm-12 col-md-4
col-sm-12 col-md-4
    
      <div class="row">
        <div class="col-sm-12 col-md-4">1</div>
        <div class="col-sm-12 col-md-4">2</div>
        <div class="col-sm-12 col-md-4">3</div>
      </div>

Here's how to have different column flows for all three device widths.

col-sm-12 col-md-6 col-lg-3
col-sm-12 col-md-6 col-lg-3
col-sm-12 col-md-6 col-lg-3
col-sm-12 col-md-6 col-lg-3
    
      <div class="row">
        <div class="col-sm-12 col-md-6 col-lg-3">1</div>
        <div class="col-sm-12 col-md-6 col-lg-3">2</div>
        <div class="col-sm-12 col-md-6 col-lg-3">3</div>
        <div class="col-sm-12 col-md-6 col-lg-3">4</div>
      </div>

Columns will be full width on small devices by default and the col-sm-12 class can be omitted if desired. Therefore, the following code will produce the same result as the example above.

    
      <div class="row">
        <div class="col-md-6 col-lg-3">1</div>
        <div class="col-md-6 col-lg-3">2</div>
        <div class="col-md-6 col-lg-3">3</div>
        <div class="col-md-6 col-lg-3">4</div>
      </div>

Changing the gutter between columns

There are 3 gutter classes which can be applied to rows: gutter-default, gutter-double, and gutter-none. If no gutter class is applied, then gutter-default will be assumed.
